Practice 5: Python RegEx and Receipt Parsing
Этот проект содержит практические задания по использованию регулярных выражений в Python.

Структура папки
receipt_parser.py — основной скрипт для парсинга данных из текстового чека.

regex_exercises.py — решение 10 дополнительных задач на использование RegEx.

raw.txt — исходный текст чека для обработки.
